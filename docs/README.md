# Documentation

- **`unity-catalog-enabled-workspace-setup.md`** — Unity Catalog enablement and workspace setup (admins).

Optional maintainer checklists and similar files are **not** stored in this repository. If you maintain the course, keep a private copy under `.local-maintainer/` on your machine (see `.gitignore`). That folder is never committed.

To **remove files from past Git history** (e.g. after deleting paths that were once public), use a history rewrite tool such as [`git filter-repo`](https://github.com/newren/git-filter-repo) with `--path` / `--invert-paths`, then force-push; coordinate with anyone else using the repo.
